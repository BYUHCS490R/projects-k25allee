Answers

1. The outside box is the parent and can control where you put the direct "kids," but they don't control the "grandkids." For example, if you want to create 3 columns or something else in a header or main tag. For both flex and grid, we need to know who/what is the parent and who/what is the child. Div tags can be used to be a new parent and separate things. For example, the parent tag will keep how big the columns are, but changes from the kids would take up their own space.
2. The 2 main differences between grid and flex are that grid is 2-dimensional (columns & rows) and flex is 1-dimensional (only columns), and with grid it tells the kids their size while flex lets the kids decide what size they want to be and then decides how the layout goes.
